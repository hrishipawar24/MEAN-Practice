Angular Interpolation

In Angular, interpolation is a powerful technique that allows you to bind data from your component's TypeScript code to your HTML template. It provides a simple way to display dynamic content within your application. â‰ 

// app.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  name = 'Angular';
}

//In html file we access that ts file data using interpolation.
<h1>Hello, {{ name }}!</h1>
